The important file in this application is server.py

To run this application, simply do

-> uvicorn server:app --host 0.0.0.0 --port 8001

-> ngrok http 8001

the main.py file is for testing


Another way to run this application for handling more than 10000 traffics, simply do

-> move to folder containing kafka

-> bin/zookeeper-server-start.sh config/zookeeper.properties

-> bin/kafka-server-start.sh config/server.properties

-> bin.kafka-topics.sh --create --topic github-pr-reviewer --bootstrap-server localhost:9092 --partitions 1 --replication-factor 1

-> uv run main_app.py

-> ngrok http 8001

In main_app.py, I used kafka for producing the message in queue and also for consuming the message from queue
